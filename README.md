# 📝 Plataforma Web de Evaluación Docente - UPGOP

¡Bienvenido al repositorio de la **Plataforma Web de Evaluación Docente**, un proyecto desarrollado como parte de mis estancias universitarias en la **Universidad Politécnica de Gómez Palacio (UPGOP)**. Esta solución digital fue diseñada para automatizar, agilizar y modernizar el proceso de evaluación del desempeño académico por parte de la comunidad estudiantil.

## 📌 Descripción del Proyecto

El proyecto consistió en el diseño e implementación de un sistema web integral que sustituye los métodos tradicionales de evaluación, centralizando la recolección de datos mediante **formularios dinámicos** y garantizando el almacenamiento seguro de los resultados en una base de datos relacional. El enfoque principal estuvo en optimizar la **experiencia de usuario (UX)** para asegurar una participación fluida, rápida y accesible para todos los estudiantes.

### 🚀 Impacto y Beneficios Clave
* **Formularios Inteligentes:** Implementación de reactivos dinámicos que se adaptan según la asignatura, tipo de docente o carrera.
* **Optimización de la UX/UI:** Interfaz limpia, intuitiva y responsiva que reduce la fricción y el tiempo que le toma al estudiante completar sus evaluaciones.
* **Almacenamiento Estructurado:** Procesamiento eficiente y seguro de las respuestas individuales, blindando la integridad y el anonimato de la información.
* **Disponibilidad Centralizada:** Acceso multiplataforma desde cualquier dispositivo conectado a la red universitaria, facilitando la administración del proceso.

---

## 🛠️ Tecnologías y Herramientas Utilizadas

* **Backend:** PHP (Programación Orientada a Objetos y manejo de sesiones seguras).
* **Base de Datos:** MySQL (Diseño relacional, llaves foráneas y optimización de consultas para reportes).
* **Frontend:** HTML5, CSS3 (Bootstrap) y JavaScript (para la validación de formularios en tiempo real y dinamismo).
* **Servidor/Entorno:** Apache / XAMPP.

---

## ✨ Características Principales

1. **Autenticación Estudiantil:** Módulo de inicio de sesión seguro que vincula al alumno con su carga académica correspondiente en el cuatrimestre vigente.
2. **Formularios Dinámicos:** Generación automática de las encuestas dependiendo del número de profesores y materias que el alumno tiene inscritos.
3. **Control de Progreso:** Indicadores visuales que muestran al estudiante qué evaluaciones ha completado y cuáles tiene pendientes.
4. **Validación en Tiempo Real:** Filtros del lado del cliente y servidor para evitar el envío de formularios incompletos o respuestas duplicadas.
5. **Panel de Almacenamiento:** Arquitectura de base de datos optimizada para consolidar miles de respuestas listas para su posterior análisis estadístico por parte del departamento de Calidad Educativa.
